# ⚡ Faulty Calculator

A fun and interactive calculator built using **HTML5, CSS3, and JavaScript**. Unlike a traditional calculator, this calculator has a **30% chance of intentionally displaying an incorrect result**, making it an entertaining project to demonstrate JavaScript logic, randomness, and DOM manipulation.

## ⚙️ How It Works

- The calculator performs normal arithmetic operations.
- Before displaying the result, a random number is generated.
- If the generated value falls within a predefined range (30% probability), the calculator intentionally displays an incorrect answer.
- Otherwise, it returns the correct mathematical result.

This project was created to practice JavaScript logic while building something fun and creative.
